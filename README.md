# Skills

Plataforma web para el aprendizaje de habilidades.

## Tecnologías
- Node.js + Express
- MongoDB
- EJS
- Nginx + PM2 (despliegue en Azure)

## Instalación
1. Clona el repositorio
2. Instala dependencias: `npm install`
3. Configura las variables de entorno en un archivo `.env`
4. Ejecuta: `npm start`
